# User Stories

## Merchant User Stories
- As a merchant, I want to sign up and complete an onboarding wizard so that I can quickly set up my business profile and start my loyalty program.
- As a merchant, I want to input my business information (name, type, location, contact) so that my profile is complete and customers can find me.
- As a merchant, I want to select and configure a loyalty program (e.g., X visits for a reward) so that I can define how customers earn rewards.
- As a merchant, I want to configure rewards (description, threshold, expiry) so that I can set up attractive incentives.
- As a merchant, I want to upload my logo and select colors/themes to customize my card layout so that it reflects my brand.
- As a merchant, I want to generate a unique QR code or link for customers to join my loyalty card so that they can easily enroll.
- As a merchant, I want to view a list of opted-in customers and basic segments so that I can understand my customer base.
- As a merchant, I want to view basic analytics (customers joined, stamps issued, rewards redeemed, repeat visit rate) so that I can measure program success.
- As a merchant, I want to integrate subscription and payment options (free trial + paid plan) so that I can manage my account billing.
- As a merchant, I want to issue stamps manually via my dashboard or validate stamps (via QR/scan) so that I can cater for special cases or in-store issuance.
- As a merchant, I want to run re-engagement campaigns (e.g., identify lapsed customers and send push notifications) so that I can bring back customers who haven’t visited recently.
- As a merchant, I want to implement gamification features (e.g., birthday rewards, scratch-and-win games) so that I can increase engagement and delight customers.

## Customer User Stories
- As a customer, I want to install the app (iOS/Android) or open the PWA so that I can access loyalty programs.
- As a customer, I want to browse or search for participating merchants near me so that I can find relevant loyalty cards.
- As a customer, I want to join a merchant’s loyalty card via QR code or link so that I can start earning stamps.
- As a customer, I want to view my digital loyalty card with stamp progress and reward status so that I can track my earnings.
- As a customer, I want to receive a stamp after a purchase (triggered by merchant) so that my progress updates automatically.
- As a customer, I want to redeem rewards once the threshold is met, showing a code or in-store redemption UI so that I can get my free item.
- As a customer, I want to receive push/in-app notifications (e.g., “Stamp earned!”, “You’re 1 away!”, “Redeem reward!”) so that I stay informed.
- As a customer, I want to manage a basic user profile and settings so that I can personalize my experience.
- As a customer, I want to collect stamps through multiple methods (tap phone, scan QR code, type code) so that I have flexibility.
- As a customer, I want to receive surprise rewards or special treats (e.g., birthday reward, scratch-and-win) so that I feel appreciated and excited.
- As a customer, I want to be reminded or re-engaged if I haven’t visited a merchant in a while so that I’m encouraged to come back.
